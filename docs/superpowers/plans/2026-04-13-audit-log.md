# Audit Log Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Extend the structured log with TurnStart, TurnEnd, ApprovalRequested, ApprovalResolved, and Error entries, and update the history explorer to display the full audit trail from the log.

**Architecture:** Five new `LogEntry` variants in ox-kernel. Write sites in `run_turn()` (turns, errors) and `ThreadNamespace` (approval events, since it routes approval writes and has access to SharedLog). History explorer reads from `log/entries` instead of `history/messages`, parsing LogEntry types directly.

**Tech Stack:** Rust, ox-kernel (LogEntry/SharedLog), ox-cli (thread_registry, parse, history_view, theme)

- [ ] **Step 1: Add a `pending_tool_name` method to get the tool name before approval clears it**

We need to read the tool name from the pending approval request before the response write clears it. The cleanest way: capture it in ThreadNamespace's write handler before routing to ApprovalStore.

```rust
impl AsyncWriter for ThreadRegistry {
    fn write(&mut self, to: &Path, data: Record) -> BoxFuture<Result<Path, StoreError>> {
        let Some((thread_id, sub)) = Self::split_thread_path(to) else {
            return Box::pin(std::future::ready(Err(StoreError::NoRoute {
                path: to.clone(),
            })));
        };
        let ns = self.ensure_mounted(&thread_id);

        if let Some(approval_sub) = Self::is_approval_path(&sub) {
            // Log approval events to the structured log
            let action = approval_sub.components.first().map(|s| s.as_str());
            match action {
                Some("request") => {
                    // Log ApprovalRequested before routing (pending not yet set, but we have the data)
                    if let Some(val) = data.as_value() {
                        let json = structfs_serde_store::value_to_json(val.clone());
                        let tool_name = json.get("tool_name")
                            .and_then(|v| v.as_str())
                            .unwrap_or("")
                            .to_string();
                        let input_preview = json.get("input_preview")
                            .and_then(|v| v.as_str())
                            .unwrap_or("")
                            .to_string();
                        let entry = serde_json::json!({
                            "type": "approval_requested",
                            "tool_name": tool_name,
                            "input_preview": input_preview,
                        });
                        let log_val = structfs_serde_store::json_to_value(entry);
                        ns.log.write(
                            &structfs_core_store::path!("append"),
                            Record::parsed(log_val),
                        ).ok();
                    }
                }
                Some("response") => {
                    // Log ApprovalResolved — read tool_name from pending before it's cleared
                    let tool_name = ns.approval.pending_tool_name();
                    if let Some(val) = data.as_value() {
                        let json = structfs_serde_store::value_to_json(val.clone());
                        let decision = json.get("decision")
                            .and_then(|v| v.as_str())
                            .unwrap_or("")
                            .to_string();
                        let entry = serde_json::json!({
                            "type": "approval_resolved",
                            "tool_name": tool_name.unwrap_or_default(),
                            "decision": decision,
                        });
                        let log_val = structfs_serde_store::json_to_value(entry);
                        ns.log.write(
                            &structfs_core_store::path!("append"),
                            Record::parsed(log_val),
                        ).ok();
                    }
                }
                _ => {}
            }
            ns.approval.write(&approval_sub, data)
        } else {
            let result = ns.write(&sub, data);
            Box::pin(std::future::ready(result))
        }
    }
}
```

```rust
// ---------------------------------------------------------------------------
// LogDisplayEntry — parsed log entry for the history explorer
// ---------------------------------------------------------------------------

/// A single log entry parsed for display in the history explorer.
#[derive(Debug, Clone)]
pub struct LogDisplayEntry {
    pub index: usize,
    /// The LogEntry type tag: "user", "assistant", "tool_call", "tool_result",
    /// "turn_start", "turn_end", "approval_requested", "approval_resolved",
    /// "error", "meta".
    pub entry_type: String,
    /// Primary display text (summary line).
    pub summary: String,
    /// Expandable detail blocks (reused from HistoryBlock for message types).
    pub blocks: Vec<HistoryBlock>,
    /// Metadata fields for rendering.
    pub meta: LogEntryMeta,
    /// Duplicate detection flags (applies to user/assistant only).
    pub flags: EntryFlags,
}

/// Type-specific metadata for rendering.
#[derive(Debug, Clone, Default)]
pub struct LogEntryMeta {
    pub scope: Option<String>,
    pub tool_name: Option<String>,
    pub tool_use_id: Option<String>,
    pub decision: Option<String>,
    pub input_preview: Option<String>,
    pub input_tokens: Option<u32>,
    pub output_tokens: Option<u32>,
    pub block_count: usize,
    pub text_len: usize,
}

/// Parse raw StructFS Values (LogEntry JSON) into display entries.
pub fn parse_log_entries(values: &[Value]) -> Vec<LogDisplayEntry> {
    let mut entries: Vec<LogDisplayEntry> = values
        .iter()
        .enumerate()
        .filter_map(|(index, val)| parse_one_log_entry(index, val))
        .collect();

    // Duplicate detection on user/assistant entries
    detect_duplicates_log(&mut entries);
    entries
}

fn parse_one_log_entry(index: usize, val: &Value) -> Option<LogDisplayEntry> {
    let map = match val {
        Value::Map(m) => m,
        _ => return None,
    };

    let entry_type = match map.get("type") {
        Some(Value::String(s)) => s.clone(),
        _ => return None,
    };

    match entry_type.as_str() {
        "user" => {
            let content = map.get("content")?;
            let scope = map.get("scope").and_then(|v| match v {
                Value::String(s) => Some(s.clone()),
                _ => None,
            });
            let (blocks, summary, block_count, text_len) = parse_content_blocks("user", content);
            Some(LogDisplayEntry {
                index,
                entry_type,
                summary,
                blocks,
                meta: LogEntryMeta {
                    scope,
                    block_count,
                    text_len,
                    ..Default::default()
                },
                flags: EntryFlags::default(),
            })
        }
        "assistant" => {
            let content = map.get("content")?;
            let scope = map.get("scope").and_then(|v| match v {
                Value::String(s) => Some(s.clone()),
                _ => None,
            });
            let (blocks, summary, block_count, text_len) =
                parse_content_blocks("assistant", content);
            Some(LogDisplayEntry {
                index,
                entry_type,
                summary,
                blocks,
                meta: LogEntryMeta {
                    scope,
                    block_count,
                    text_len,
                    ..Default::default()
                },
                flags: EntryFlags::default(),
            })
        }
        "tool_call" => {
            let name = get_string(map, "name").unwrap_or_default();
            let id = get_string(map, "id");
            let input = map.get("input").map(format_value);
            let scope = get_string(map, "scope");
            let summary = format!("tool_call: {name}");
            let blocks = vec![HistoryBlock {
                block_type: "tool_use".to_string(),
                text: None,
                tool_name: Some(name.clone()),
                tool_use_id: id.clone(),
                input_json: input,
            }];
            Some(LogDisplayEntry {
                index,
                entry_type,
                summary,
                blocks,
                meta: LogEntryMeta {
                    scope,
                    tool_name: Some(name),
                    tool_use_id: id,
                    block_count: 1,
                    ..Default::default()
                },
                flags: EntryFlags::default(),
            })
        }
        "tool_result" => {
            let id = get_string(map, "id");
            let output = match map.get("output") {
                Some(Value::String(s)) => Some(s.clone()),
                Some(v) => Some(format_value(v)),
                None => None,
            };
            let scope = get_string(map, "scope");
            let text_len = output.as_ref().map(|s| s.len()).unwrap_or(0);
            let summary = format!(
                "tool_result{}",
                id.as_ref()
                    .map(|i| format!(": {}...", &i[..i.len().min(12)]))
                    .unwrap_or_default()
            );
            let blocks = vec![HistoryBlock {
                block_type: "tool_result".to_string(),
                text: output,
                tool_name: None,
                tool_use_id: id.clone(),
                input_json: None,
            }];
            Some(LogDisplayEntry {
                index,
                entry_type,
                summary,
                blocks,
                meta: LogEntryMeta {
                    scope,
                    tool_use_id: id,
                    text_len,
                    block_count: 1,
                    ..Default::default()
                },
                flags: EntryFlags::default(),
            })
        }
        "turn_start" => {
            let scope = get_string(map, "scope");
            Some(LogDisplayEntry {
                index,
                entry_type,
                summary: "turn start".to_string(),
                blocks: vec![],
                meta: LogEntryMeta {
                    scope,
                    ..Default::default()
                },
                flags: EntryFlags::default(),
            })
        }
        "turn_end" => {
            let scope = get_string(map, "scope");
            let input_tokens = match map.get("input_tokens") {
                Some(Value::Integer(n)) => Some(*n as u32),
                _ => None,
            };
            let output_tokens = match map.get("output_tokens") {
                Some(Value::Integer(n)) => Some(*n as u32),
                _ => None,
            };
            let summary = match (input_tokens, output_tokens) {
                (Some(i), Some(o)) => format!("turn end ({i}in / {o}out)"),
                _ => "turn end".to_string(),
            };
            Some(LogDisplayEntry {
                index,
                entry_type,
                summary,
                blocks: vec![],
                meta: LogEntryMeta {
                    scope,
                    input_tokens,
                    output_tokens,
                    ..Default::default()
                },
                flags: EntryFlags::default(),
            })
        }
        "approval_requested" => {
            let tool_name = get_string(map, "tool_name").unwrap_or_default();
            let input_preview = get_string(map, "input_preview").unwrap_or_default();
            let summary = format!("{tool_name}: \"{input_preview}\"");
            Some(LogDisplayEntry {
                index,
                entry_type,
                summary,
                blocks: vec![],
                meta: LogEntryMeta {
                    tool_name: Some(tool_name),
                    input_preview: Some(input_preview),
                    ..Default::default()
                },
                flags: EntryFlags::default(),
            })
        }
        "approval_resolved" => {
            let tool_name = get_string(map, "tool_name").unwrap_or_default();
            let decision = get_string(map, "decision").unwrap_or_default();
            let summary = format!("{tool_name}: {decision}");
            Some(LogDisplayEntry {
                index,
                entry_type,
                summary,
                blocks: vec![],
                meta: LogEntryMeta {
                    tool_name: Some(tool_name),
                    decision: Some(decision),
                    ..Default::default()
                },
                flags: EntryFlags::default(),
            })
        }
        "error" => {
            let message = get_string(map, "message").unwrap_or_default();
            let scope = get_string(map, "scope");
            Some(LogDisplayEntry {
                index,
                entry_type,
                summary: message,
                blocks: vec![],
                meta: LogEntryMeta {
                    scope,
                    ..Default::default()
                },
                flags: EntryFlags::default(),
            })
        }
        "meta" => {
            let data = map.get("data").map(format_value).unwrap_or_default();
            let summary = if data.len() > 80 {
                format!("{}...", &data[..80])
            } else {
                data
            };
            Some(LogDisplayEntry {
                index,
                entry_type,
                summary,
                blocks: vec![],
                meta: LogEntryMeta::default(),
                flags: EntryFlags::default(),
            })
        }
        _ => None,
    }
}

fn get_string(map: &std::collections::BTreeMap<String, Value>, key: &str) -> Option<String> {
    match map.get(key) {
        Some(Value::String(s)) => Some(s.clone()),
        _ => None,
    }
}

/// Duplicate detection for log entries — only applies to user/assistant types.
fn detect_duplicates_log(entries: &mut [LogDisplayEntry]) {
    for i in 1..entries.len() {
        let et = &entries[i].entry_type;
        if et != "user" && et != "assistant" {
            continue;
        }
        if entries[i].meta.text_len == 0 {
            continue;
        }
        for j in (0..i).rev() {
            if entries[j].entry_type != entries[i].entry_type {
                continue;
            }
            let text_i = concat_text(&entries[i].blocks);
            let text_j = concat_text(&entries[j].blocks);
            if !text_i.is_empty() && text_i == text_j {
                entries[i].flags.duplicate_content = true;
                entries[i].flags.duplicate_of = Some(j);
            }
            break;
        }
    }
}
```

```rust
fn render_message_entry(
    lines: &mut Vec<Line>,
    entry: &LogDisplayEntry,
    is_selected: bool,
    _is_expanded: bool,
    theme: &Theme,
) {
    let cursor = if is_selected { ">" } else { " " };
    let role_style = match entry.entry_type.as_str() {
        "user" => theme.history_role_user,
        "assistant" => theme.history_role_assistant,
        "tool_call" => theme.history_role_tool,
        "tool_result" => theme.history_role_tool,
        _ => theme.history_meta,
    };

    let role_label = match entry.entry_type.as_str() {
        "tool_call" => "tool_call",
        "tool_result" => "tool_result",
        other => other,
    };

    let mut spans = vec![
        Span::styled(
            cursor.to_string(),
            if is_selected { theme.history_selected } else { theme.history_summary },
        ),
        Span::styled(format!("#{:<3} ", entry.index), theme.history_index),
        Span::styled(format!("{:<12}", role_label), role_style),
    ];

    if entry.flags.duplicate_content {
        spans.push(Span::styled(
            format!("[DUP of #{}] ", entry.flags.duplicate_of.unwrap_or(0)),
            theme.history_duplicate,
        ));
    }

    let summary = if entry.summary.len() > 60 {
        format!("{}...", &entry.summary[..60])
    } else {
        entry.summary.clone()
    };
    spans.push(Span::styled(
        format!("\"{summary}\""),
        if is_selected { theme.history_selected } else { theme.history_summary },
    ));

    if entry.meta.block_count > 0 {
        spans.push(Span::styled(
            format!(
                " ({} block{}, {} chars)",
                entry.meta.block_count,
                if entry.meta.block_count == 1 { "" } else { "s" },
                entry.meta.text_len
            ),
            theme.history_meta,
        ));
    }

    lines.push(Line::from(spans));
}
```
